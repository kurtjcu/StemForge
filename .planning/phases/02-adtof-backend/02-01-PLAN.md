    ```python
    """ADTOF-pytorch automatic drum transcription backend.

    Provides :class:`AdtofBackendProtocol` (structural interface for ADT backends)
    and :class:`AdtofBackend` (ADTOF Frame-RNN implementation).

    Usage::

        backend = AdtofBackend()
        backend.load(device="cuda")
        events = backend.predict(Path("drums.wav"))
        backend.evict()
    """
    from __future__ import annotations

    import logging
    import pathlib
    from typing import Protocol, runtime_checkable

    import torch

    from utils.errors import ModelLoadError
    from utils.midi_io import NoteEvent

    logger = logging.getLogger(__name__)
    ```

    Define `AdtofBackendProtocol`:
    ```python
    @runtime_checkable
    class AdtofBackendProtocol(Protocol):
        """Structural interface for automatic drum transcription backends."""

        def load(self, device: str = "cpu") -> None: ...
        def predict(self, audio_path: pathlib.Path) -> list[NoteEvent]: ...
        def evict(self) -> None: ...
    ```

    Define `AdtofBackend`:
    ```python
    class AdtofBackend:
        """ADTOF Frame-RNN drum transcription backend.

        Implements :class:`AdtofBackendProtocol` via structural subtyping
        (no inheritance required).
        """

        def __init__(self) -> None:
            self._model: torch.nn.Module | None = None
            self._device: str = "cpu"
    ```

    Implement `load(self, device: str = "cpu") -> None`:
    - Import adtof_pytorch functions at call time (inside load, not module top level) to preserve lazy loading: `from adtof_pytorch import create_frame_rnn_model, calculate_n_bins, load_pytorch_weights, get_default_weights_path`
    - Wrap entire body in try/except, catching any Exception and re-raising as `ModelLoadError(str(e), model_name="adtof-drums")`
    - Call `n_bins = calculate_n_bins()`
    - Call `model = create_frame_rnn_model(n_bins)`
    - Call `model.eval()` immediately after creation
    - Call `weights_path = get_default_weights_path()`
    - Call `model = load_pytorch_weights(model, str(weights_path), strict=False)`
    - Call `model.to(device)`
    - Store `self._model = model` and `self._device = device`
    - Log: `logger.info("ADTOF model loaded on %s", device)`

    Implement `evict(self) -> None`:
    - If `self._model is not None`, move to CPU: `self._model.cpu()`
    - Set `self._model = None`
    - If device was CUDA: `torch.cuda.empty_cache()`
    - Log: `logger.info("ADTOF model evicted")`

    Add a placeholder `predict` method that raises `NotImplementedError("predict() implemented in Plan 02-02")` -- this is needed so the Protocol structural check works (method must exist). Plan 02 will replace this.
